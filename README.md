## Стек технологий
- **Backend**: Flask
- **Database**: Supabase (PostgreSQL)
- **Документирование**: Visio, Word

## Модули экзамена

### Модуль 1: Проектирование ER-диаграммы 
Спроектировать ER-диаграмму для информационной системы управления производством и реализацией продукции.

**Требования:**
- 3-я нормальная форма (3NF)
- Ссылочная целостность
- Первичные и внешние ключи
- Согласованное именование
- Формат: PDF

### Модуль 2: Разработка БД
Создать базу данных на основе ER-диаграммы в Supabase.

**Требования:**
- Таблицы основных сущностей
- Ограничения и связи
- Импорт данных из `Заказчики.json`

### Модуль 3: Создание запроса
Запрос для вычисления полной стоимости заказа:
- Количество продукции
- Стоимость материалов с учетом нормы расхода

### Модуль 4: Разработка информационной системы (Flask)
- Форма авторизации с ролями (Администратор, Пользователь)
- Интерактивная капча (пазл)
- Блокировка после 3 ошибок
- Админ-панель управления пользователями
- Валидация логина/пароля

### Модуль 5: Проектная документация
Описание функционального назначения и методов с параметрами.

### Модуль 6: Интеграция и тестирование
- Валидация данных клиентов
- Тест-кейсы в Word
- Эмулятор данных (TransferSimulator.exe)
